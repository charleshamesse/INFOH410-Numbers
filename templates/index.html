Recognizing Numbers

Using Convolutional Neural Networks

1. Train

Before being able to recognize any handwritten number, the neural network need first to be trained. This is done by supervised learning where we give the neural network an image and then compare what he thought it was with the solution. Then the error will be back propagated to change tha valus of each neurons. A dataset of 40000 images will be used to train this network. This database comes from the NIST.

To begin the training click the button underneath. A graph will show the training of the network in real time.
